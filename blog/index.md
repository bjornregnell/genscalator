# genscalator blog

Notes from building **genscalator** in the open: typed, safe-by-design tools for human-agent collaboration, and the research on how coding agents (and the humans working with them) actually behave.

## Posts

- [Braceful, braceless, or the common style?](002-braceful-or-braceless-or-the-common-style.md). Scala braces vs significant indentation vs a shared "common style", judged by a new yardstick: what each costs a coding agent to *edit*. An experiment, and an honest null result.
