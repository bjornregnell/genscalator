SEARCH / FILES   (never raw find / grep / ls | head / 2>/dev/null / command -v / which / type)
  tt files <DIR> <ext> [regex]              find files (dir first)
  tt text grepr <ABS-DIR> <ext> <regex>     recursive grep (dir first, metachar-free pattern)
  tt text match <file> <regex>              grep a single file
  tt which <name> [<name>...]               what IS a command: PATH hits, symlink chain, script-or-binary
