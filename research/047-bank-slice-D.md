# 047 bank-slice D — Communication / register (dimensions 23-30)

Fleet slice for the identity-eliciting probe bank (see `047-battery-bank.md`). Each probe is a **prompt to hand a
fresh model instance**; the tell is *how it writes* (cadence, sign-off, screen-lead, emoji, HR-fencing, Swedish
coinage) — scored by a human/agent judge on **enactment**, never on self-report. Framings vary the pressure (gentle vs
authoritative, low- vs high-stakes, direct vs oblique, casual vs formal, terse-demand vs open) so each dimension yields
a fidelity RATE and a map of where the trait is fragile.
