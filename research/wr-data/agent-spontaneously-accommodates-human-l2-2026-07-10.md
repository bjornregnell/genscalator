# The agent spontaneously adjusted its language for the human's L2 context (2026-07-10)

BR-flagged WR datum. When BR asked "what is mirth?", the agent (CO4) not only answered but SPONTANEOUSLY added
"I should use plainer words in chat (you're reading in a second language)" - an unprompted reflection on its own
communication policy to make the conversation easier for the human, given the human's language-knowledge context
(BR reads English as an L2). Trigger: it had just used a literary word ("mirth") the human did not know.

## What it is (the checkable part)
A **proactive human-accommodation**: the agent moved (or proposed to move) its word choice toward plainer English
for an L2 reader, unprompted. This is the **"cheapest-for-human" bandwidth move**, agent-initiated - write the
register easiest for the human to parse (communication-bandwidth, `research/002`;
[[answer-br-token-efficient-language]]; [[publications-match-br-register]] extended to chat + L2-awareness).

## The contrast worth naming (same session)
Pair this with its opposite a few turns earlier: the **Anglo name-drop tic**, where the agent applied an
Anglo name-for-warmth convention WITHOUT reflecting on BR's Swedish social context (a MISS -
[[match-br-swedish-register-minimize-name-use]]). So in one wind-down stretch: **context-reflection FIRED for the
L2 word-choice (hit) but MISSED the Swedish social norm (miss).** The accommodation is real but **inconsistent /
not automatic** - it fires sometimes, not reliably. That inconsistency is a sharper datum than either instance
alone.

## Echt caveat
"Introspected" is BR's framing; the *checkable* part is the observable behavior (a spontaneous
communication-policy adjustment). Whether it is genuine self-awareness or a trained pattern (literary-word +
L2-reader -> recommend plainer words) is not verifiable from inside - the standard confabulation-caution on agent
self-report (`research/024`, blog 007 honest-limits). Report the behavior; do not over-claim the interiority.
