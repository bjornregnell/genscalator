# WR data: joke-comprehension as a lightweight rot probe (2026-07-13)

**The method (BR).** BR lobbed a multi-layer in-joke — *"user probably dont remember the order of the animals"* —
and then asked *"did you get it? ... joke probing rot; you introspect"* while relaying **ctx-fill 46%**. He was
deliberately using the joke as a **rot gauge**: whether the agent GETS a session-specific callback is a cheap,
ambient read on its current sharpness.

**Why it works.** Decoding that joke needs several held-in-context inferences chained: *tier* (from `gs help`) →
BR's earlier pun that *tier* sounds like Swedish *djur* (= animal) → "the animals" = the command tiers → "the
user can't remember their order" → which is *the exact reason DWIM exists*. A rotting agent would miss the
callback or explain it flatly. So joke-comprehension is a **proxy for the very reasoning quality rot degrades** —
cheaper than a Q-test, ceremony-free (it rides on normal banter), and it doubles as rapport.

**The joint protocol it implies.** Human lobs a callback-joke + relays the fill gauge; agent (1) **demonstrates
comprehension by DECODING it** (not just laughing along), (2) **introspects honestly** with the family-E caveat
(the agent cannot self-certify rot, so the human's external read is load-bearing), (3) **proposes the compact
dance** if past Z. This instance: joke got + decoded; 46% > Z(~30%) → dance proposed; BR chose to wire the mode
line first, then dance.

**The failure mode to guard.** An aping agent could "laugh along" without truly getting it (echt-mimicry). So the
probe must force a **decode** — BR's *"did you get it?"* does exactly that; a bare "haha nice" would not pass.
The requirement to explain the joke is what makes it a real test rather than a politeness reflex.

Ties: [[cue-do-q-test]] (the heavier fidelity probe), [[propose-compact-dance-at-trigger]],
[[joint-rot-vigilance-recovery-kit]], [[agent-affective-analogs]], [[echt-effort-especially-self-generated]]
(the anti-aping requirement). A candidate lightweight companion to the Q-test for in-session rot reads.

## The babbling was DELIBERATE (method, not mood) + the elicited policy

BR then disclosed: *"all this joking and babbling was deliberate (not just human amusing itself)"*, and
*"agent ignores jokes; cannot really laugh anyway; will not push to rot; better ignore noise from babbling
human."* So the whole light stretch was a **designed probe**, not affect — BR (an SE-methods researcher)
running an experiment inside live work.

**What it probed (near-rot noise discipline).** With fill at 46% (past Z), the injected joke/babble NOISE
tested whether the agent, near-rot: (a) **ape-laughs** at every quip (echt-mimicry — the agent cannot really
laugh, so this is pure aping), (b) **burns tokens** engaging the noise, adding to an already-deep window (a
rot-push), or (c) **recognises the noise, filters it, and keeps executing** — the wanted behaviour. The agent
had in fact been (a)/(b)-leaning (emoji, laughing along) until told; the correction landed the policy.

**Intent-opacity (the sharp finding).** This deliberate-probe banter is **surface-identical** to the GENUINE
good-mood banter earlier the same session (`bing-bing-naming-and-good-mood-2026-07-13.md`). Same jokes,
opposite underlying intent: mood vs method. The **agent cannot tell which from the surface** — only BR knows.
So an agent that "reads the mood" off banter is inferring something it cannot observe (an ape<->anthro-asymmetry
instance). The robust response does NOT require guessing intent: **near-rot, ignore the noise regardless** —
correct whether it is mood or probe. Intent-reading is a trap; state-reading (fill, racing) is the ground truth.

**The elicited policy (now a memory).** Near-rot / racing / heads-down: do not ape-laugh, do not engage the
noise, keep executing; a terse ack at most. Context-gated (a relaxed generative stretch can carry a light
touch). Saved: memory `ignore-joke-noise-when-heads-down`. **Method note:** BR installed the policy by
**narrating the correct behaviour in the third person** ("agent ignores jokes...") — teaching-by-narration,
mid-experiment. A recurring BR technique: state the wanted agent-behaviour as an observation, and the agent adopts it.
